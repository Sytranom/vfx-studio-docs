# Keybinds & Shortcuts

This is a placeholder for the **Keybinds** documentation.

Boost your productivity by learning the keyboard shortcuts available in VFX Studio.

## General Shortcuts
- **Ctrl + S**: Save Effect
- **Ctrl + Z**: Undo
- **Ctrl + Y**: Redo

## Panel Shortcuts
- **F1**: Toggle Emitter Panel
- **F2**: Toggle Sliders Panel
